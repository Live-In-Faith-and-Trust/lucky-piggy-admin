## Google AdMob 광고

### 광고 유형
| 유형 | 설명 |
|------|------|
| 보상형 광고 (Rewarded) | 시청 완료 시 티켓 지급. 미션·출석·보물캐기 등에서 활용 |
| 네이티브 광고 (Native) | 피드형 배너 광고 |

### 광고 사이클
- 일정 횟수 광고 시청 시 럭키포켓(광고 사이클 보너스) 지급
- 출석 시 하루 2회 광고 추가 보상 가능

### Mock 지원
- `packages/infra`에 Mock 광고 클래스 완비 (dev 환경에서 실제 광고 없이 테스트)

## 어드민 관련 기능

### 푸시 알림
- **공지 발송**: 전체/특정 유저 그룹에 푸시 발송
- **예약 발송**: 특정 시각에 발송 예약
- **발송 이력**: 과거 발송 내역 및 수신 통계

### 광고 설정
- **광고 단가/비율 모니터링**: AdMob 수익 확인 (AdMob 콘솔 연동 또는 참고용)
- **광고 보상 설정**: 보상형 광고 시청 시 지급 티켓 수 조정 (미션 설정과 연동)
- **광고 사이클 보너스 설정**: 럭키포켓 지급 기준 광고 횟수 조정
