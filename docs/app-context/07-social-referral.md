# 소셜 · 친구초대 · 쿠폰

## 친구초대 (Referral)

### 구조
- 각 유저는 고유한 `referralCode` 보유
- 신규 가입 시 초대코드 입력하면 `referredBy`에 초대자 코드 저장
- 초대 성공 시 **초대자** + **피초대자** 모두 보상 티켓 지급

### 공유 방식
- 딥링크 공유 (앱 링크)
- 카카오톡 공유
- Play Install Referrer (앱 설치 유입 추적)

---

## 쿠폰 코드 (Coupon Redeem)

- 관리자가 발급한 쿠폰 코드 입력
- 코드 유효 시 티켓 지급 (타입: `coupon`)
- 1인 1회 or 다회 사용 정책

---

## 어드민 관련 기능

### 친구초대
- **레퍼럴 통계**: 초대 링크를 통한 가입자 수, 보상 지급 현황
- **레퍼럴 트리**: 특정 유저의 초대 계보 조회
- **보상 지급 현황**: referral 타입 티켓 트랜잭션 조회

### 쿠폰
- **쿠폰 코드 생성**: 코드 문자열, 보상 티켓 수, 사용 횟수 제한, 만료일 설정
- **쿠폰 목록 관리**: 활성/만료/사용완료 상태 관리
- **쿠폰 사용 이력**: 특정 쿠폰을 사용한 유저 목록
- **대량 쿠폰 생성**: 이벤트용 단일 사용 코드 배치 생성
